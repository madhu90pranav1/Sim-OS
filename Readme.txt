The project has the following files in it:

1. Sourcecode ( project4sourcecode.cpp)
2. Readme file (Readme.txt)
3. Configuration file ( config.sys)
4. Sample program file (sampleprogram1.txt)


The project can be compiled and executed by the following steps:
 
 Step 1. Create a directory by typing the following command:
 
 {cslinux1:~} mkdir project3

 Step 2: Enter the directory by typing the following command:
 
 {cslinux1:~} cd project3

 
Step 3: Open  a text editor , copy contents of the source code and save the file as project3sourcecode.cpp

Step 4: Open a text editor and copy the contents of config.sys and save it as  config.sys

Step 5: Open a text editor and copy the contents of the file containing the sample program and save it sampleprogram1.txt
Step 6. compile the project by typing the following command in the shell:

{cslinux1:~/project3} g++ -g project4sourcecode.cpp -o simos -pthread

The above command will generate an executable file with the name " simos"

Step 7. Run the project by typing the following:

{cslinux1:~/project3} ./simos

When the program is running the following propmpt will appear:

command>

Step 8. Enter the folllowing command

command>s
 The filename will be asked . Enter the sampleprogram1.txt and press enter.
The sampleprogram1 is submitted and ready for execution . The program will return to command>

Any other command can be then tried out.
